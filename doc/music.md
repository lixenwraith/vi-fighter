# Architecture Overview

```
┌─────────────────────────────────────────────────────────────────┐
│                         MusicSystem                             │
│  (ECS System - handles events, drives Sequencer)                │
└─────────────────────┬───────────────────────────────────────────┘
                      │
┌─────────────────────▼───────────────────────────────────────────┐
│                         Sequencer                               │
│  - Tempo clock (BPM → samples/step)                             │
│  - Step position tracking                                       │
│  - Pattern transition orchestration                             │
├─────────────────────┬───────────────────────────────────────────┤
│      BeatTrack      │              MelodyTrack                  │
│  - Pattern lookup   │  - Pattern lookup                         │
│  - DrumVoice pool   │  - TonalVoice pool (8 max)                │
│  - Crossfade logic  │  - Voice stealing                         │
└─────────┬───────────┴──────────────────┬────────────────────────┘
          │                              │
┌─────────▼──────────┐    ┌──────────────▼────────────────────────┐
│    DrumVoice       │    │           TonalVoice                  │
│  - Instrument type │    │  - Instrument type                    │
│  - One-shot gen    │    │  - Note/freq state                    │
│  - Position cursor │    │  - ADSR envelope                      │
└────────────────────┘    └───────────────────────────────────────┘
```
